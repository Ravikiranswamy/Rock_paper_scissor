# Rock, Paper, Scissors Game
This project is a simple implementation of the classic game "Rock, Paper, Scissors" using HTML, CSS, and JavaScript.

# How to Play
Clone the Repository:

Simply open the java.html file in your preferred web browser (Chrome, Firefox, Safari, etc.).

Start Playing:

Click on your choice of "Rock", "Paper", or "Scissors".

The computer will randomly select its choice.

The result (win, lose, or draw) will be displayed on the screen.

# Features

Responsive Design: The game is designed to work on desktop and mobile browsers.

Random Computer Choice: The computer's choice (Rock, Paper, or Scissors) is randomly generated.

Winning Logic: Implements the standard rules of Rock, Paper, Scissors.

# File Structure

java.html: Main HTML file that contains the structure of the game.

java.css: CSS file for styling the game.

java.js: JavaScript file that implements the game logic.
